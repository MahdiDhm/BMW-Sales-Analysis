# BMW Sales Analysis

Analysis of BMW sales data from 2010 to 2024 with interactive Power BI dashboard.

## Overview

This project analyzes BMW automotive sales trends over 14 years, providing insights into market performance and sales patterns through an interactive dashboard.

## Repository Content

- **BMW_sales_data_2010-2024.csv**: Sales data spanning 2010-2024
- **BMW_Sales_Dashboard.pbix**: Interactive Power BI dashboard

## Key Features

- Historical sales trends
- Performance metrics and analytics
- Interactive visualizations

### Analyze the Data

The CSV file contains raw sales data for custom analysis with your preferred tools (Python, Excel, SQL, etc.).
